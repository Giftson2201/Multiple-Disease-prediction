# 🩺 Multiple Disease Prediction System

An AI-powered healthcare web application that predicts multiple diseases using Machine Learning models.  
Built with Python and Streamlit, this system helps users perform early-stage disease risk assessment based on medical input parameters.

---

## 🚀 Features

✅ Diabetes Prediction  
✅ Heart Disease Prediction  
✅ Parkinson’s Disease Prediction  
✅ Interactive Streamlit Web Interface  
✅ Real-time Predictions  
✅ Trained Machine Learning Models  
✅ User-Friendly Healthcare Dashboard

---

## 🧠 Machine Learning Models Used

| Disease | Algorithm Used |
|---|---|
| Diabetes Prediction | Support Vector Machine (SVM) |
| Heart Disease Prediction | Logistic Regression |
| Parkinson’s Prediction | Support Vector Machine (SVM) |
